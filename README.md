# i-SEAL GitHub Pages Multipage Website

This package is a simple static multipage website for GitHub Pages.

## Included files
- index.html
- professor.html
- research.html
- members.html
- publications.html
- news.html
- contact.html
- styles.css
- script.js

## Upload
Upload all files to the root of the repository.

## Publish
Go to Settings → Pages → Deploy from a branch → main → /(root).
